# Hello World

## Objective

Create a Python script that prints "Hello, World!" to the console.

## Requirements

1. Create a file at `src/hello.py`
2. The script should print exactly: `Hello, World!`
3. The output should end with a newline

## Expected Output

When running `python src/hello.py`, the output should be:

```
Hello, World!
```

## Notes

- This is a simple verification task
- No input is required
- No external dependencies should be used
