# Task: Hello World

## Objective

Create a Python program that prints "Hello, World!" to the console.

## Requirements

1. Create a file at `src/hello.py`
2. The program should print exactly: `Hello, World!`
3. The program should exit with code 0

## Success Criteria

- Running `python src/hello.py` outputs `Hello, World!`
- Exit code is 0
